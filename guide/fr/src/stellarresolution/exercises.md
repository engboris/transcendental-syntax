# Exercices

Donner le résultat de l'exécution pour les constellations ci-dessous.
Vous pouvez vérifier en écrivant le code suivant dans un fichier :

```
print <constellation>
```

puis en l'exécutant.

Par exemple :

```
print X +f(X); @-f(a).
```

1. `@+f(X); -f(a).`
2. `@+f(X); -f(Y) a.`
3. `@+f(X) X; -f(a); -f(b).`
4. `@+1(X) -2(X); -2(X) +3(X).`
5. `@-1(X) +2(X); -2(X) +1(X).`
6. `@-1(X) +2(X); -2(X) +1(X).`
7. `@-f(X) X; +f(+g(a)); -g(X) X.`
